# Daily Command Brief

`/brief` is a decision-oriented view of persisted active recommendations. It shows sync freshness, recommendation generation time, a primary focus, next actions, campaign/event/resource sections, review items, and items outside the selected planning-duration categories.

Available-play-time choices reorder structural categories; they are not measured durations and do not imply energy costs. The strategic objective influences scoring only for an exact selected entity relationship.

The optional Reflection section is disabled by default, local-only, and isolated from recommendation scoring. It uses short curated ESV references and original reflections without external retrieval.
